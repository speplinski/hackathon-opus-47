Recording workflow

  • Open scene in browser → F11 for fullscreen
  • Start ScreenStudio / OBS recording on browser window (1080p60)
  • Reload page (Cmd+R) — 1.5s pre-roll, then animation auto-plays
  • Stop recording after the scene's full duration + 1s safety
  • In post: cut start to first frame of motion, sync to corresponding voiceover beat

Signature visual language

  • Brygada 1918 italic — same as the website hero
  • Cream #EDE8DC on near-black #14130F
  • Clip-path wipe from left, ~1s reveal, cubic-bezier easing
  • Coral #E17B5D as one accent beat per scene — never decorative
  • Hard cuts between scenes in post; never dissolves
